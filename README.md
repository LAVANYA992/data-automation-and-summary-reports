# Data Cleaning & Reporting Automation using Python and Power BI

## Project Overview

This project automates the process of cleaning, preprocessing, and reporting historical sales data using Python and Power BI. The workflow handles missing values, duplicate records, inconsistent data formats, and generates an interactive dashboard for business reporting and decision-making.

## Objective

The primary objective of this project is to automate data cleaning and reporting workflows to improve data quality, reduce manual effort, and provide meaningful business insights through interactive visualizations.

## Project Workflow

1. Imported the historical sales dataset.
2. Inspected the dataset structure and data types.
3. Identified and handled missing values.
4. Removed duplicate records.
5. Standardized text fields and corrected inconsistent formatting.
6. Converted date columns into datetime format.
7. Created a new Profit Margin feature.
8. Generated an automated summary report.
9. Exported the cleaned dataset.
10. Built an interactive Power BI dashboard using the cleaned data.

## Data Cleaning Process

The following preprocessing techniques were applied:

- Missing value detection and handling
- Duplicate record removal
